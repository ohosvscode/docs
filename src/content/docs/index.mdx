## 正在进行的项目

<CardGrid stagger>
	<Card title="vscode ArkTS 插件" icon="puzzle">
		一个 VSCode 插件，旨在为 `.ets` 语言和 `hvigor` 项目提供基础语言功能支持，如代码补全、高亮、错误提示、引用跳转等。[点击前往仓库](https://github.com/Groupguanfang/arkTS)  [点击查看文档](/arkts/install)
	</Card>
	<Card title="Electron for HarmonyOS" icon="seti:react">
		这是一个模版项目, 旨在让更多人了解如何把`Electron`应用程序移植到`HarmonyOS`上。[点击前往仓库](https://github.com/ohosvscode/ohos_electron_hap)
	</Card>
  <Card title="Hvigor Project Detector" icon="magnifier">
    基于 `Rust` 写的项目检测工具，指定一个基础工作目录，扫描此目录下所有鸿蒙工程。并提供 NAPI 绑定到TypeScript。[点击前往仓库](https://github.com/ohosvscode/project-detector)
  </Card>
  <Card title="OpenHarmony SDK Downloader" icon="download">
    一个用于下载 OpenHarmony SDK 的工具, 支持下载指定API版本和指定架构的 SDK，并提供 `github action` 直接用于你的 CI/CD 流水线中。[点击前往仓库](https://github.com/ohosvscode/sdk-downloader)
  </Card>
  <Card title="Twoslash for ArkTS" icon="seti:shell">
    [twoslash](/twoslash/guide) 的 `ArkTS` 语言支持，用于在文档中预先生成丰富类型信息的标记。[点击前往仓库](https://github.com/ohosvscode/twoslash)
  </Card>
  <Card title="Visual Studio Code on HarmonyOS PC" icon="vscode">
    (开发中...) 运行在鸿蒙 PC 上的 `Visual Studio Code`, 目前仍然在开发中，详见 [仓库](https://github.com/ohosvscode/vscode)。
  </Card>
</CardGrid>
